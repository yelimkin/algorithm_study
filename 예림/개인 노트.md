> ### 바꾸고 싶은 타입.valueOf(타입을 바꾸고 싶은 값)
>  바꾸고 싶은 타입(Integer, Float, Double, String 등)으로 값을 바꾸어 반환한다.
> ```
> String str = String.valueOf(5);
> System.out.println(str);
> 
> -> "5"
> ```

> ### "String".matches(정규 표현식)
>   문자열 "String"이 주어진 정규 표현식과 일치하는지에 따라 참/거짓 값을 반환한다.
>   원하는 패턴의 문자열이 있는지 알아볼 때, 유용하다.
> ```
> boolean test = "java String test".matches(".*java.*");
> System.out.println(test);
>
> -> true
> ```
> - 자바 정규 표현식 : https://adjh54.tistory.com/104

> ### "String".repeat(반복할 횟수) -> "StringStringString..."
>   문자열 "String"을 반복할 횟수 만큼 반복한 문자열을 반환한다.
>   반복할 횟수가 0이면 빈 문자열을 반환하고 음수이면 IllegalArgumentExceptionthrow 에러를 반환한다.
>   내부적으로 배열의 모든 값을 같은 값으로 초기화하는 Arrays.fill() 및 배열을 복사하는 System.arraycopy()를 호출하여 새 문자열을 만든다.
>   - Arrays.fill(초기화할 배열, 초기화할 값)
>   - System.arraycopy(원본 배열 a, 원본 배열 a의 복사 시작 위치, 복사된 배열 b, 복사될 배열 b의 복사 시작 위치, 원복 배열 a의 복사될 요소 개수)
